# Operations

> Runtime layer. Everything that keeps the brain running.

- `workflows/` — processor prompts and scheduled workflows
- `connectors/` — MCP and poller connections feeding `.inbox/`
- `skills/` — bundled Obsidian plugins and Claude skills
- `schedule.md` — calendar view of all scheduled runs
- `runs/` — append-only run log per month
- `upstream/` — pinned copies of files from prior-art projects
